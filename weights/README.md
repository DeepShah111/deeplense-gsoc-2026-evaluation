# Pre-trained Model Weights

Due to GitHub's 100MB file size limit, the trained PyTorch `.pth` files for the baseline, transfer learning, and Vision Transformer models are hosted externally.

To run the inference and ensemble notebooks (`05_Inference_Ensemble_and_TTA.ipynb`) without retraining, download the weights from the secure Google Drive links below and place them directly into this directory:

* **ViT-B/16 Champion (330MB):** https://drive.google.com/file/d/1tRYLZsebrUNrLil2kbaXUHC_uoFvJId2/view?usp=sharing
* **ResNet-18 Robust (42MB):** https://drive.google.com/file/d/1m5JNGUV8gVnYOEwDWCB_7uLhOjggSNf0/view?usp=sharing
